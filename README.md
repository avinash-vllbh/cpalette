# Cpalette

CPalette
----------------------

A simple color gem that returns you with a palette of colors which are web friendly and easy to use.

## Installation

Add this line to your application's Gemfile:

    gem 'cpalette'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cpalette
